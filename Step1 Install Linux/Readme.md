
##**สรุปขั้นตอนการติดตั้ง (CentOS 7)**

( บูตจากแผ่นหรือไฟล์ ISO และเลือก "Install CentOS 7 )

1\.ตั้งค่าภษา (แนะนำภาษาอังกฤษ) 

2\.เลือก Zone

3\.เลือกซอฟต์แวร์ที่ต้องการ เป็น Server with GUI  และ เลือก Packet 

4\.จัดการพาร์ติชัน (HDD 20 GB):

`  `Swap		 กำหนดเป็น 2 เท่าของแรม

`  `/tmp 		ขนาดประมาณ 1GB

` `/home  	ขนาดประมาณ 5GB

` `/var   		ขนาดประมาณ 5GB

` `/        		พื้นที่ที่เหลือทั้งหมด

5\. Add user
